Beste bezoeker, welkom op mijn website!

Mijn naam is Joren De Rijk, ik ben een 18-jarige student aan het Immaculata instituut in Oostmalle. Op deze school volg ik de richting IT en Netwerken. Na mijn middelbare studies zou ik mij graag willen verdiepen in autotechnologie, doordat ik hier in mijn vrije tijd al veel mee bezig ben. Wanneer ik werk aan auto’s en motors ervaar ik een enorm groot gevoel van vrijheid. Daarom droom ik van een job in deze sector. Mijn website is opgedeeld in verschillende tabs, een voor elk vak dat deel uitmaakt van mijn studierichting. Op de hoofdpagina kan u alle hoogtepunten van mijn jaar terugvinden. De specifieke taken zijn opgedeeld per vak. Deze kan u vinden in de navigatiebalk onder de tab “vakken”. Er is ook een tabblad aangemaakt met quotes die mij persoonlijk aanspreken en zaken die ik heb geleerd doorheen mijn schoolcarrière op het Immaculata instituut.

Hopende dat u alles goed terugvindt op mijn website, wens ik u veel kijk- en leesplezier toe!

2019-09-30

Ik kreeg op een persoonlijke rondleiding door de afdelingen en men overliep met mij hoe ze bonnen en facturen moesten opstellen. We voerde kleine reparaties en herstellingen uit. Ik maakte ook deel uit van een wekelijkse teammeeting met het team van deze afdeling. Ook telefonisch werden klanten op de hoogte gesteld van de status van hun reparatie. Tijdens mijn verblijf mocht ik een lokale laptop gebruiken voor mijn GIP-taken, dit toestel heb ik persoonlijk geprept en toegevoegd aan het domein voor eigen gebruik.

2019-10-01

Vandaag heb ik geholpen bij het opstellen van een html5 template voor een persoonlijk project van mijn stagebegeleider. Hierna moest ik 5 Hp Probook's domain joinen en gebruiksklaar maken. Ook zijn er paar kleine reparaties uitgevoerd.

2019-10-02

Mijn dag begon met het vervangen van een batterij van een Iphone 7plus, ook een paar reparaties uitgevoerd zoals een laptop met een bureaublad error. Hierna kreeg ik een beknopte uitleg hoe ik een juiste diagnose moet stellen van een defect toestel.

Op 3 Oktober moest ik een diagnose stellen van een laptop met een slechte connector van het scherm (het scherm viel in bepaalde posities weg). Ook werd er een grafische kaart vervangen van een acer nitro. In de namiddag moest ik een batterij van een Ipad Mini vervangen. Voor mijn eerste case-study ging ik mee op verplaatsing voor het installeren van printers. Eenmaal terug bij VanRoey Automation heb ik met zelfgevonden linux software een laptop kunnen herstellen waarvan de eigenaar het wachtwoord vergeten was.

2019-10-04

Op mijn laatste dag begon ik met een laptop die koelproblemen had. Dit was makkelijk op te lossen doordat de ventilator defect bleek te zijn na diagnose. Ook ben ik mensen gaan ondervragen voor aanvullende informatie voor mijn Gip-taken.

La foire aux questions.

Vous trouverez ici les questions les plus fréquemment posées concernant le programme Teams. Si vous avez des questions plus spécifiques, vous pouvez nous contacter en ligne.

Comment changer sa disponibilité dans le programme Teams?

Dans le coin supérieur droit du menu principal du programme, cliquez sur son avatar. On voit alors plusieurs options, dont la "disponibilité". En cliquant dessus, vous aurez la possibilité de le modifier.

Comment pouvez-vous ajouter du matériel pédagogique à votre équipe?

On navigue vers son équipe personnelle et on passe à l'onglet "fichiers". Sous la barre principale, vous verrez plusieurs options, dont "upload". Si vous cliquez dessus, les fichiers souhaités peuvent être téléchargés.

Comment installer l'application mobile de Teams?

Cela dépend de l'appareil mobile utilisé. Pour les utilisateurs d'Android, naviguez vers le Play Store. Saisissez le terme de recherche "Équipes" dans la barre de recherche. L'application qui sera alors affichée doit être installée.

Pour un utilisateur Apple, rendez-vous sur l'App Store.

Saisissez le terme de recherche "Équipes" dans la barre de recherche. L'application qui sera alors affichée doit être installée. Après l'installation, vous pouvez vous connecter avec vos données de connexion personnelles.

Je n'arrive pas à faire démarrer mes équipes. Et maintenant?

Tout d'abord, le programme est vérifié pour voir s'il est toujours actif dans la gestion des tâches. Il peut être ouvert via l'explorateur Windows. Si vous voyez toujours le programme Équipes dans cette liste, il devrait être fermé. Après cela, il peut être redémarré. Si cela ne résout pas le problème, la réinstallation du programme est également une option.
