# FAQ

Voor frans moesten we een FAQ schrijven.

##Vous trouverez ici les questions les plus fréquemment posées concernant le programme Teams. Si vous avez des questions plus spécifiques, vous pouvez nous contacter en ligne.


##Comment changer sa disponibilité dans le programme Teams ?
Dans le coin supérieur droit du menu principal du programme, cliquez sur son avatar.
On voit alors plusieurs options, dont la "disponibilité".
En cliquant dessus, vous aurez la possibilité de le modifier.


##Comment pouvez-vous ajouter du matériel pédagogique à votre équipe ?
On navigue vers son équipe personnelle et on passe à l'onglet "fichiers".
Sous la barre principale, vous verrez plusieurs options, dont "upload".
Si vous cliquez dessus, les fichiers souhaités peuvent être téléchargés.


##Comment installer l'application mobile de Teams ?
Cela dépend de l'appareil mobile utilisé.
Pour les utilisateurs d'Android, naviguez vers le Play Store.
Saisissez le terme de recherche "Équipes" dans la barre de recherche.
L'application qui sera alors affichée doit être installée.

##Pour un utilisateur Apple, rendez-vous sur l'App Store.
Saisissez le terme de recherche "Équipes" dans la barre de recherche.
L'application qui sera alors affichée doit être installée.
Après l'installation, vous pouvez vous connecter avec vos données de connexion personnelles.


##Je n'arrive pas à faire démarrer mes équipes. Et maintenant ?
Tout d'abord, le programme est vérifié pour voir s'il est toujours actif dans la gestion des tâches.
Il peut être ouvert via l'explorateur Windows. 
Si vous voyez toujours le programme Équipes dans cette liste, il devrait être fermé.
Après cela, il peut être redémarré.
Si cela ne résout pas le problème, la réinstallation du programme est également une option.
